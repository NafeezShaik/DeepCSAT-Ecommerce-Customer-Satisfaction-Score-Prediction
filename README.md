# DeepCSAT-Ecommerce-Customer-Satisfaction-Score-Prediction
DeepCSAT: E-Commerce Customer Satisfaction Score Prediction using Deep Learning (ANN) &amp; Traditional ML. Built on 85K+ support interactions with full EDA, hypothesis testing, feature engineering (TF-IDF, SMOTE, VIF), 7 models (LR, DT, RF, XGBoost, LightGBM, ANN), and a Streamlit deployment app.
